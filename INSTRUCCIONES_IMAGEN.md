# Instrucciones para Guardar la Imagen del Edificio ISMA

## Pasos a seguir:

1. **Guarda la imagen adjunta del Edificio Empresarial ISMA** que enviaste en el chat.

2. **Renombra la imagen** como: `edificio-isma.jpg`

3. **Copia la imagen** a la siguiente carpeta:
   ```
   d:\ESCRITORIO\InmobiliariaApp\properties\static\img\
   ```

4. La ruta completa del archivo debe ser:
   ```
   d:\ESCRITORIO\InmobiliariaApp\properties\static\img\edificio-isma.jpg
   ```

## Alternativa: Copia manual

Si prefieres hacerlo manualmente:
1. Descarga la imagen del edificio que adjuntaste en el chat
2. Navega a: `d:\ESCRITORIO\InmobiliariaApp\properties\static\img\`
3. Pega la imagen allí y renómbrala como `edificio-isma.jpg`

## Verificación

Una vez guardada la imagen, la landing page la mostrará automáticamente en la sección "Visítanos en el Edificio Empresarial ISMA".

---

**Nota**: La aplicación ya está configurada para usar esta imagen. Solo falta guardarla en la ubicación correcta.
